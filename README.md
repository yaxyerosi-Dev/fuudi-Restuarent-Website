# fuudi-Restuarent-Website

A modern and responsive restaurant landing page built using HTML, CSS, Tailwind CSS, and animations. This project showcases a beautiful food website UI with sections for menu items, categories, testimonials, services, and footer links.

🚀 **Features**
Responsive Navigation Bar
Modern Hero Section
Food Categories Section
Standout Menu Cards
Customer Testimonials
Services Section
Animated UI Elements
Attractive Footer Design
Smooth Hover Effects & Animations
Mobile Responsive Layout
🛠️ Technologies Used
HTML5
CSS3
Tailwind CSS
Alpine.js
Google Fonts
